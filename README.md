# Omega Vault

Omega Vault is an offline-first collection management platform designed for reliability, portability, and long-term scalability.

The first available module, **Omega Comics**, provides comic inventory tracking, seller tools, and optional Pro features.

---

## Platform Overview

- Offline-first architecture
- Local data storage by default
- Optional Pro subscription features
- Modular structure (future modules supported)

---

## Omega Comics Module

- Comic inventory management
- Collection tracking
- Seller tools & listing exports
- Market and graph tools (Pro)
- Device-based Pro approval system

---

## Subscription Model

- Free tier supports up to 20 items
- Pro unlocks advanced features
- Pro requires account verification
- Approved device limits apply
- Verification required on or before expiration date
- No grace period after expiration
- Data access and exports are always available
